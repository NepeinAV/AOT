Запуск программы

Просто откройте файл ./build/index.html

Функционал программы

  • Выделение слов из текста
    • Исходный код располагается в файле ./src/classes/Words/index.js
    Методы:
    • breakText(text: string): Array<string>
      : разбивает текст на слова, попутно удаляя нелексические элементы.
    • goodSymbol(symbol: integer): boolean
      : выполняет проверку символа на вхождение в диапазон допустимых символов.
    • hyphen(word: string, i: integer): boolean
      : возвращает true, если вокруг дефиса разрешённые символы.
    • irrationalNumber(text: string, pos: integer): boolean
      : позволяет выяснить, является ли запятая в тексте десятичным разделителем.
  • Стемминг слов
    • Исходный код располагается в файле ./src/classes/Stemmer/index.js
    Методы:
    • getRV(word: string): Array<string>
      : выделяет RV часть слова.
    • getR1(word: string): Array<string>
      : выделяет R1 часть слова.
    • getR2(r1: string): Array<string>
      : выделяет R2 часть слова.
    • stemWords(words: string): Array<string>
      : выполняет стемминг всех слов в переданном ему массиве.
    • stemWord(word: string): string
      : выполняет стемминг слова.
    • findEnding(endings: Array, word: string, has: boolean): string|boolean
      : выполняет поиск окончания в слове
    • firstStep(word: string): string
      : выполняет первый шаг алгоритма стемминга
    • secondStep(word: string): string
      : выполняет второй шаг алгоритма стемминга
    • thirdStep(word: string): string
      : выполняет третий шаг алгоритма стемминга
    • fourthStep(word: string): string
      : выполняет четвёртый шаг алгоритма стемминга
  • Составление ПОД
    • Исходный код располагается в файле ./src/classes/Descriptors/index.js
    Методы:
    • countDescriptors(words: Array<string>): Object{key: string, value: integer}
      : подсчитывает дескрипторы.
    • cleanDescriptors(descriptors: Object{key: string, value: integer}): void
      : удаляет слова, присутствущие в стоп-листе.
    • cleanDescriptors(descriptors: Object{key: string, value: integer}): Object{key: string, value: integer}
      : возвращает первые двадцать дескрипторов.
    • getDescriptors(words: Array<string>): void
      : объединяет в себе методы countDescriptors и cleanDescriptors.
  • Поиск N-грамм
    • Исходный код располагается в файле ./src/classes/NGram/index.js
    Методы:
    • countNGrams(words: Array<string>, n: integer): Object{key: string, value: integer}
      : подсчитывает n-граммы.
  • Составление ПОЗ и вычисление релевантности для проиндексированных документов
    • Исходный код располагается в файле ./src/classes/Search/index.js
    Методы:
    • handleRequest(req: string): Array<string>
      : обрабатывает текст запроса путём разбиения на слова и последующего их стемминга.
    • calcWeights(poz: Array<string>): Array<integer>
      : вычисляет весовые коэффициенты слов поискового запроса.
    • calculateR(poz: Array<string>, weights: Array<integer>): Object{key: string, value: integer}
      : рассчитывает релевантность проиндексированных документов.
  • Сортировка содержимого выходного экрана
    • Исходный код располагается в файле ./src/classes/Sort/index.js
    Методы:
    • swap(a: integer, n: integer): void
      : выполняет перестановку элементов в массиве
    • isort(compareFn: function, left: integer, right: integer): Array<string|integer|Array>
      : сортировка вставками, использующаяся тогда, когда в массиве меньше 33 элементов.
    • qsort(compareFn: function, left: integer, right: integer): Array<string|integer|Array>
      : быстрая сортировка
